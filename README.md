# 💰 FinFlow

A modern full-stack personal finance management application that helps users track income, expenses, savings goals, and overall financial progress through a clean and intuitive dashboard.

---

## ✨ Features

- 🔐 Secure user authentication with JWT
- 💰 Track income and expenses
- 🎯 Create and monitor savings goals
- 📊 Interactive financial dashboard
- 📈 Expense analytics and monthly trends
- 📋 View recent transactions and goals
- ❤️ Life Vault for storing important financial memories
- 🔍 Search transactions and goals
- 📱 Responsive design for desktop and mobile

## 🚀 Installation

```bash
git clone https://github.com/puppalaleelaprasanna5-ux/finflow.git

cd finflow

npm install

npm run dev
```

Open:

```
http://localhost:3000
```
